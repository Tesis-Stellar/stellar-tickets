# Stellar Tickets — Frontend

Aplicación **React + Vite + TypeScript** (Tailwind, shadcn/ui) para el catálogo Web2 y la capa Freighter/Soroban.

## Desarrollo

Desde esta carpeta:

```sh
npm install
npm run dev
```

El servidor de desarrollo usa el puerto **8080** por defecto (`vite.config.ts`).
